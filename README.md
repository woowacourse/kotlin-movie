# kotlin-movie

## 1단계 구현 기능

### Movie
- [x] 영화 제목을 가진다.
- [x] 러닝 타임을 가진다.

### Schedule
- [ ] 영화 시작 시간을 가진다.
- [ ] 영화 종료 시간을 가진다.
- [ ] 상영 영화 정보를 가진다.

### SeatNumber
- [x] 행을 가진다.
- [x] 열을 가진다.

### Seat
- [ ] 좌석 번호를 가진다.
- [ ] 좌석 등급을 가진다.

### Reservation
- [ ] 스케줄을 가진다.
- [ ] 선택된 좌석들을 가진다.

### Cart
- [ ] Reservation를 가진다.

### Payment
- [ ] 결제 수단 별 할인율을 가진다.

### DiscountPolicy
- [ ] 할인 정책에 맞게 할인된 금액을 계산한다.

### PointPolicy
- [ ] 포인트 사용 정책에 맞게 금액을 계산한다.
