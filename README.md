# kotlin-movie

## 기능 목록

### 영화

- [ ] 영화 제목을 가진다
- [ ] 영화 상영 길이(러닝타임)를 가진다
- [ ] 영화 상영 기간(시작일 ~ 종료일)을 가진다

### 상영 정보

특정 영화가 어느 상영관에서 언제 상영되는지에 대한 스케줄을 관리합니다.

- [ ] 영화를 가진다
- [ ] 상영 날짜 및 시작 시간을 가진다 (`LocalDateTime`)
- [ ] 상영관을 가진다
- [ ] 해당 상영 회차의 예약 가능 좌석 목록을 관리한다
    - 상영관에 속한 좌석을 기반으로 예약 상태와 함께 초기화한다

### 상영관

- [ ] 상영관 이름을 가진다
- [ ] 상영관 운영 시간 가진다
- [ ] 상영관에 속한 예약 가능한 좌석 목록을 가진다.

### 4. 좌석

상영관 내 개별 좌석의 정보를 관리합니다.

- [ ] 좌석의 위치를 가진다
- [ ] 좌석의 등급을 가진다

### 5. 예약 상태를 가진 좌석

- [ ] 좌석을 가진다
- [ ] 예약 상태를 가진다 (`available` (예약 가능) 또는 `reserved`)

### 위치

- [ ] 행과 열을 가진다
    - 행: 알파벳, 열: 숫자

### 등급

- [ ] 금액을 가진다

### 할인 정책 관리

모든 할인 정책은 `DiscountPolicy` 인터페이스를 구현합니다.

#### 결제 수단 할인 (`PaymentDiscountPolicy`)

- [ ] 카드 결제 시 할인을 적용한다
- [ ] 현금 결제 시 할인을 적용한다

#### 조기/시간 할인 (`EventDiscountPolicy`)

- [ ] 무비데이 조건에 해당하는 경우 할인을 적용한다
- [ ] 특정 시간대 조건에 해당하는 경우 할인을 적용한다
