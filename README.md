# 2단계 - 영화 예매(콘솔)

## 기능 목록

## ReservationController
### 예매 시작
- [ ] 새 예매 생성 여부를 입력받는다. (Y/N)
  - [ ] Y/N이 아닌 값이 입력되면 다시 입력받는다.

### 영화 선택
- [ ] 예매할 영화 제목을 입력받는다.
  - [ ] 존재하지 않는 영화 제목이면 다시 입력받는다.

### 날짜 선택
- [ ] 날짜를 입력받는다. (YYYY-MM-DD)
  - [ ] 형식이 올바르지 않으면 다시 입력받는다.

### 상영 시간 선택
- [ ] 해당 날짜에 선택한 영화의 상영이 없으면 에러를 반환한다.
- [ ] 해당 날짜의 상영 목록을 번호와 함께 출력한다.
- [ ] 상영 번호를 입력받는다.
  - [ ] 유효하지 않은 번호이면 다시 입력받는다.
  - [ ] 선택한 상영 시간이 장바구니의 기존 예매와 겹치면 안내 메시지를 출력하고 다시 입력받는다.

### 좌석 선택
- [ ] 좌석 배치도를 좌석 등급(S/A/B)과 예약 상태와 함께 출력한다.
- [ ] 예약할 좌석을 입력받는다. (쉼표로 구분, 예: A1, B2)
  - [ ] 형식이 올바르지 않으면 다시 입력받는다.
  - [ ] 해당 상영관에 존재하지 않는 좌석이면 다시 입력받는다.
  - [ ] 이미 예약된 좌석이면 다시 입력받는다.
  
---
## CartController
### 장바구니
- [ ] 선택한 영화, 날짜, 시간, 좌석 정보를 장바구니에 추가한다.
- [ ] 장바구니에 추가된 내역을 출력한다.
- [ ] 다른 영화를 추가할지 여부를 입력받는다. (Y/N)
  - [ ] Y이면 영화 선택 단계로 돌아간다.
  - [ ] N이면 장바구니 전체 내역을 출력하고 결제 단계로 진행한다.

---
## PaymentController

### 포인트 사용
- [ ] 사용할 포인트를 입력받는다. (없으면 0)
  - [ ] 보유 포인트보다 큰 값이면 다시 입력받는다.

### 결제 수단 선택
- [ ] 결제 수단을 선택받는다. (1: 신용카드 5% 할인, 2: 현금 2% 할인)
  - [ ] 유효하지 않은 값이면 다시 입력받는다.

### 가격 계산
- [ ] 좌석 등급별 가격을 합산한다.
- [ ] 무비데이 할인(10%)을 적용한다.
- [ ] 시간 할인(2,000원)을 적용한다.
- [ ] 포인트를 차감한다.
- [ ] 결제 수단 할인을 적용한다.
- [ ] 최종 결제 금액을 출력한다.

### 결제 확인
- [ ] 최종 금액으로 결제할지 여부를 입력받는다. (Y/N)
  - [ ] Y이면 예매를 완료하고 내역을 출력한다.
  - [ ] N이면 예매를 취소한다.

## 단위 테스트 목록

### 좌석(Seat)
- [x] 열이 한 글자 대문자 알파벳이면 좌석이 정상 생성된다.
- [x] 열이 한 글자 대문자 알파벳이 아니면 예외가 발생한다.
- [x] 행이 양수이면 좌석이 정상 생성된다.
- [x] 행이 0이면 예외가 발생한다.
- [x] 행이 음수이면 예외가 발생한다.

### 좌석등급(SeatGrade)
- [x] S석의 가격은 18,000원이다.
- [x] A석의 가격은 15,000원이다.
- [x] B석의 가격은 12,000원이다.

### 사용자(User)
- [x] ID는 양수일 때 정상 생성 되어야 한다.
- [x] ID가 음수일 때 예외가 발생한다.
- [x] 포인트 사용 시 포인트가 차감된다.
- [x] 보유 포인트보다 많은 금액을 차감하면 예외가 발생한다.

### 할인정책(DiscountPolicy)
- [x] 매월 10일에 상영되는 영화는 10% 할인된다.
- [x] 매월 20일에 상영되는 영화는 10% 할인된다.
- [x] 매월 30일에 상영되는 영화는 10% 할인된다.
- [x] 무비데이가 아닌 날에는 무비데이 할인이 적용되지 않는다.
- [x] 오전 11시 이전에 시작하는 상영은 2,000원이 할인된다.
- [x] 오후 8시 이후에 시작하는 상영은 2,000원이 할인된다.
- [x] 오전 11시 ~ 오후 8시 사이에 시작하는 상영은 시간 할인이 적용되지 않는다.
- [x] 신용카드 결제 시 5% 할인된다.
- [x] 현금 결제 시 2% 할인된다.

### 계산기(Calculator)
- [x] 무비데이에 상영되는 영화는 10% 할인이 적용된다.
- [x] 할인 시간에 상영되는 영화는 2000원 할인이 적용된다.
- [x] 무비데이도 아니고 할인 시간도 아닌 영화는 할인이 적용되지 않는다.
- [x] 무비데이 할인과 시간 할인이 순서대로 적용된다.
- [x] 포인트가 예매 금액에서 직접 차감된다.
- [x] 결제 수단에 따라 할인이 적용된다.

### 예매(Reservation)
- [x] 영화의 id가 존재하지 않으면 예외가 발생한다.
- [x] 영화의 id가 존재할 경우, 해당 영화를 반환한다.
- [x] 예매 영화와 예매 날짜를 입력하였을 때 해당하는 상영 일정을 반환한다.
- [x] 예매 영화와 예매 날짜를 입력하였을 때 해당하는 상영 일정이 없을 경우, 예외가 발생한다.
- [x] 존재하지 않는 사용자를 찾는 경우, 예외가 발생한다.
- [x] 동일한 시간대에 이미 예매한 내역이 있으면 예외가 발생한다.
- [x] 좌석 정보가 "(알파벳)(숫자)" 형식이 아니면 예외가 발생한다.
- [x] 해당 상영관에 존재하지 않는 좌석이면 예외가 발생한다.
- [x] 이미 예약된 좌석이면 예외가 발생한다.
