### domain
- [ ] 좌석
  - [ ] 좌석은 행과 열로 구성된다.
  - [ ] 각 좌석은 등급이 매겨진다.
    - [ ] S석은 18,000원이다.
    - [ ] A석은 15,000원이다.
    - [ ] B석은 12,000원이다.
- [ ] 할인 정책
  - [ ] 10일, 20일, 30일은 10% 무비데이 할인이 적용된다.
  - [ ] 오전 11시 이전 / 오후 8시 이후 상영 건은 조조/심야 2천원 할인이 적용된다.
  - [ ] 무비데이 할인과 조조 심야 할인이 동시 적용될 시 무비데이 할인이 먼저 적용된다.
  - [ ] 포인트
    - [ ] 포인트가 없으면 사용할 수 없다.
    - [ ] 보유 포인트가 결제 금액보다 많을 시, 결제 금액 한도 내에서 사용할 수 있다.
    - [ ] 보유 포인트가 결제 금액보다 적을 시, 포인트 한도 내에서 사용할 수 있다.
  - [ ] 결제
    - [ ] 카드 결제 선택 시 총액 기준 5% 할인된다.
    - [ ] 현금 결제 선택 시 총액 기준 2% 할인된다.
- [ ] 상영
  - [ ] 서로 겹치는 시간대의 상영 건은 예매할 수 없다.
  - [ ] 이미 예약된 좌석은 추가로 예약할 수 없다.
- [ ] 계정
  - [ ] 초기 포인트 2000원이 주어진다.
