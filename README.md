# 기능 구현 사항

## 1. 영화 시간 배정

### 영화(Movie)
- [x] 러닝 타임이 같아도 id가 다르면 다른 영화로 판단한다.
- [x] 러닝 타임이 달라도 id가 같으면 같은 영화로 판단한다.

### 러닝타임(Running Time)
- [x] 러닝 타임이 0 이하일 경우 예외를 던진다.
- [x] 러닝 타임이 양수라면 해당하는 값이 저장된다.
- [x] 러닝 타임의 값이 서로 같다면 동등하다.

### 상영관(Screen)
- [x] 운영 기간이 같더라도 id가 다르면 다른 상영관으로 판단한다.
- [x] 운영 기간이 다르더라도 id가 같으면 같은 상영관으로 판단한다

### 기간(DateTimeRange)
- [x] 시작 시간이 종료 시간보다 느린 시간이라면 예외가 발생한다.
- [x] 시작 시간이 종료 시간과 같은 시간이라면 예외가 발생한다.
- [x] 2025.12.25일 12시 30분은 2025.12.25 12시 ~ 2025.12.25 13시의 기간에 포함된다.
- [x] 2025.12.25일 13시 30분과 11시 30분은 2025.12.25 12시 ~ 2025.12.25 13시의 기간에 포함되지 않는다.
- [x] 2025년 12월 25일 12시는 2025년 12월 25일 12시 ~ 13시의 기간에 포함된다
- [x] 2025년 12월 25일 13시는 2025년 12월 25일 12시 ~ 13시의 기간에 포함된다

### 배정된 영화 상영 일정 (ScheduledScreen)
- [x] 상영관의 운영시간보다 영화 상영 일정(ScheduledScreen)이 늦게 배정되면 예외를 발생시킨다.
- [x] 상영관의 운영시간보다 영화 상영 일정(ScheduledScreen)이 일찍 배정되면 예외를 발생시킨다
- [x] 영화, 상영관, 기간이 같은 일정은 동일한 일정으로 판단한다.
- [x] 영화, 상영관, 기간 중 하나만 다르더라도 다른 일정으로 판단한다
- [x] 상영관에 배정된 시간의 길이가 영화의 러닝타임과 일치하지 않으면 에외를 발생시킨다. 
- [x] 상영관에 배정된 시간의 길이가 영화의 러닝타임과 일치하면 에외를 발생시키지 않는다.
 
### 배정된 영화 상영 일정 목록 (ScheduledScreenGroup)
- [ ] 특정 영화를 요청받으면 해당하는 영화 전체 일정(MovieSchedule)을 반환한다.
