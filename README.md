# kotlin-movie

## 기능 목록

### 1. 영화

- [x] 영화 제목을 가진다
- [x] 영화 상영 길이를 가진다
- [x] 영화 상영 기간을 가진다

### 1-1. 영화 제목

- [x] 제목을 가진다

### 1-2. 상영 길이

- [x] 분 단위의 상영길이를 가진다

### 1-3. 상영 기간

- [x] 시작일과 종료일을 가진다

### 2. 좌석

- [x] 행과 열을 가진다
    - 행은 알파벳(A~J)으로 이루어져 있다
    - 열은 숫자로 이루어져 있다(1~12)
- [x] 예약 상태를 가진다
- [x] 좌석의 등급을 안다

### 3. 등급

- [x] 등급에 대한 금액을 가진다
    - S석: 18_000
    - A석: 15_000
    - B석: 12_000

### 4. 상영관

- [x] 상영관 이름을 가진다
- [x] 상영관 운영 시간 가진다
- [x] 상영관에 속한 예약 가능한 좌석 목록을 가진다.


### 상영 정보

- [x] 영화를 가진다
- [x] 상영 날짜 및 시작 시간을 가진다 (`LocalDateTime`)
- [x] 상영관을 가진다
- [x] 해당 상영 회차의 예약 가능 좌석 목록을 관리한다
    - 상영관에 속한 좌석을 기반으로 예약 상태와 함께 초기화한다

### 할인 정책 관리

모든 할인 정책은 `DiscountPolicy` 인터페이스를 구현합니다.

#### 결제 수단 할인 (`PaymentDiscountPolicy`)

- [ ] 카드 결제 시 할인을 적용한다
- [ ] 현금 결제 시 할인을 적용한다

### 이벤트 할인 (`EventDiscountPolicy`)

- [ ] 무비데이 조건에 해당하는 경우 할인을 적용한다
- [ ] 특정 시간대 조건에 해당하는 경우 할인을 적용한다

## 테스트 목록

### 영화 테스트

- [x] 영화 제목이 공백일 경우 예외를 던진다
- [x] 영화의 상영 길이가 0이하일 경우 예외를 던진다
- [x] 종료일이 시작일보다 앞 설 경우 예외를 던진다
- [x] 영화는 제목, 상영 길이, 상영 기간을 가진다

### 좌석

- [x] 행과 열을 가진다
- [x] 좌석의 초기 예약 상태는 예약 가능한 상태이다
- [x] 좌석 위치에 따른 등급을 안다
- [x] 행이 A~J 범위가 아닐 경우 예외를 던진다
- [x] 행이 알파벳이 아닌 문자가 포함될 경우 예외를 던진다
- [x] 열이 1~12 범위가 아닐 경우 예외를 던진다

### 좌석 등급

- [x] 등급에 대한 가격을 가진다
- [x] 행과 열에 대한 등급을 반환한다

### 상영관

- [x] 이름, 운영시간, 좌석 목록을 가진다
- [x] 이름이 공백일 경우 예외를 던진다
- [x] 운영 종료 시간이 시작 시간보다 앞설 경우 예외를 던진다
- [x] 좌석이 없을 경우 예외를 던진다

### 상영 정보

- [x] 영화, 상영관, 상영 날짜 및 시간을 가진다
- [x] 상영 시간이 상영관의 운영시간에 포함되지 않을 경우 예외를 던진다
- [x] 좌석을 예약 시 좌석의 상태가 바뀐다
