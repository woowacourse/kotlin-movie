## 🚀 1단계 - 영화 예매

안드로이드 8기 레벨 1미션, 영화 예매 시스템의 도메인을 관리하는 프로젝트입니다.

## 1단계 기능 및 작업 명세

### 금액 및 포인트 구현

금액을 포장하는 값 객체를 구현한다. 0 미만의 금액은 허용하지 않는다. 포인트도 같은 방식으로 포장하며, 음수가 될 수 없고 금액으로 변환할 수 있다.

### 영화 정보 값 객체 구현

상영 길이를 분 단위로 포장한다. 0 이하는 허용하지 않는다. 상영 가능 기간은 시작일과 종료일로 표현하며, 시작일이 종료일보다 이후일 수 없다. 특정 날짜가 기간 안에 포함되는지 확인할 수 있다. 좌석 번호는 행(A~E)과 열(1~4)의 조합으로 표현하며 유효 범위를 검증한다. 좌석 등급은 S, A, B 세 가지이며 각 등급의 가격은 각각 18,000원, 15,000원, 12,000원이다.

### 영화 및 좌석 구현

영화는 제목, 상영 길이, 상영 가능 기간을 가진다. 좌석은 좌석 번호와 등급만을 가지는 순수 값 객체다. 예약 여부와 같은 가변 상태를 보유하지 않는다.

### 좌석 목록 구현

좌석 목록을 일급 컬렉션으로 관리한다. 좌석 번호로 특정 좌석을 찾을 수 있고, 이미 예약된 좌석 번호 집합을 제외한 예약 가능 좌석 목록을 반환할 수 있다. 선택된 좌석들의 등급 가격 합계를 계산하며, 좌석 번호 목록을 반환할 수 있다.

### 상영관 구현

상영관은 상영관 이름과 물리적 좌석 레이아웃을 가진다.

### 상영 회차 구현

상영 회차는 특정 영화, 시작 시각, 상영관을 가진다. 종료 시각은 생성자 파라미터로 받지 않으며, 영화의 상영 길이로부터 계산된다. 해당 회차에서 예약된 좌석 번호를 관리하며, 좌석 예약과 예약 가능 좌석 조회를 담당한다.

좌석을 예약할 때는 중복 선택, 이미 예약된 좌석, 존재하지 않는 좌석을 모두 검증한다. 검증을 통과하면 예약 객체를 반환한다. 예약 객체를 생성하는 유일한 진입점이다.

### 상영 회차 목록 구현

상영 회차 목록을 일급 컬렉션으로 관리하며 순회할 수 있다. 목록이 비어 있는지 확인할 수 있고, 영화와 날짜 기준으로 필터링할 수 있다.

### 예약 구현

예약은 상영 회차와 선택된 좌석 목록을 보관한다. 영화 제목, 시작 시각, 예매 기본 가격을 제공한다.

### 예약 목록 구현

예약 목록을 일급 컬렉션으로 관리하며 순회할 수 있다. 빈 상태로 시작할 수 있다. 예약을 추가할 때 기존 예약과 상영 시간이 겹치는지 검증하며, 겹치면 예외가 발생한다. 추가 시 새로운 목록을 반환하는 불변 방식으로 동작한다.

### 영화 목록 및 스케줄러 구현

영화 목록을 일급 컬렉션으로 관리한다. 전체 목록에 특정 영화 포함 여부를 확인하고, 제목으로 영화를 검색할 수 있다.

스케줄러는 모든 상영 회차 정보를 내부에 보유한다. 영화와 날짜를 받아 해당 상영 회차 목록을 반환하고, 전체 영화 목록도 제공한다.

### 할인 정책 구현

할인 정책 sealed 인터페이스를 정의한다. 무비데이 할인 정책은 매월 10일, 20일, 30일에 10% 비율로 할인한다. 시간 할인 정책은 오전 11시 이전 또는 오후 8시 이후(정각 포함) 상영 시 2,000원을 할인한다.

### 결제 수단 구현

결제 수단 sealed 인터페이스를 정의한다. 신용카드는 5%, 현금은 2% 추가 할인을 적용한다.

### 결제 시스템 구현

할인 정책 적용 → 포인트 차감 → 결제 수단 할인 순서로 최종 결제 금액을 계산한다. 할인 정책은 리스트 순서대로 적용하며, 무비데이 → 시간 순서를 호출부에서 보장한다. 결제 결과는 최종 결제 금액과 사용된 포인트를 담은 객체로 반환한다.

## 테스트 명세

### 단위 테스트

#### 금액 및 포인트

- 초기 금액이 올바르게 생성된다
- 금액이 0 미만이면 예외가 발생한다
- 금액이 0 미만일 때 올바른 예외 메시지가 출력된다
- 포인트가 음수이면 예외가 발생한다
- 포인트를 금액으로 올바르게 변환한다

#### 영화 정보 값 객체

- 상영 길이가 0 이하이면 예외가 발생한다
- 시작일이 종료일보다 이후이면 예외가 발생한다
- 특정 날짜가 상영 가능 기간에 포함되면 true를 반환한다
- 특정 날짜가 상영 가능 기간에 포함되지 않으면 false를 반환한다
- 유효한 행과 열로 좌석 번호가 생성된다
- 행이 유효 범위를 벗어나면 예외가 발생한다
- 열이 유효 범위를 벗어나면 예외가 발생한다
- S등급의 가격은 18,000원이다
- A등급의 가격은 15,000원이다
- B등급의 가격은 12,000원이다

#### 좌석 및 좌석 목록

- 동일한 좌석 번호와 등급을 가진 두 좌석은 동등하다
- 등급이 다른 두 좌석은 동등하지 않다
- 선택한 좌석들의 등급 가격이 올바르게 합산된다
- 예약된 좌석 번호를 제외한 나머지 좌석을 반환한다
- 존재하지 않는 좌석을 찾으면 예외가 발생한다

#### 영화 목록

- 영화 목록이 비어있으면 예외가 발생한다
- 영화 목록에 찾는 영화가 있으면 true를 반환한다
- 영화 목록에 찾는 영화가 없으면 false를 반환한다
- 제목으로 영화를 찾을 수 있다
- 존재하지 않는 제목으로 찾으면 null을 반환한다

#### 상영 회차

- 종료 시각은 시작 시각에 영화 상영 길이를 더한 값이다
- 시작 시각과 상영 날짜가 올바르게 반환된다
- 두 상영 회차의 시간이 겹치면 true를 반환한다
- 두 상영 회차의 시간이 겹치지 않으면 false를 반환한다
- 이미 예약된 좌석을 다시 예약하면 예외가 발생한다
- 한 예약 요청 내에서 중복된 좌석을 선택하면 예외가 발생한다
- 존재하지 않는 좌석을 예약하면 예외가 발생한다
- 정상 예약 후 예약 가능 좌석 조회 시 해당 좌석이 제외된다
- 예약 결과의 기본 가격이 올바르게 계산된다

#### 예약 목록

- 빈 상태로 시작할 수 있다
- 시간이 겹치는 상영을 예매하면 예외가 발생한다
- 시간이 겹치지 않는 상영을 예매할 수 있다

#### 스케줄러

- 해당 영화의 상영 회차가 없으면 빈 상영 회차 목록을 반환한다
- 영화와 날짜에 맞는 상영 회차 목록을 반환한다
- 전체 영화 목록을 반환한다

#### 할인 정책

- 10일, 20일, 30일에 각각 10% 할인이 적용된다
- 해당하지 않는 날짜에는 할인이 적용되지 않는다
- 오전 11시 이전 상영에 2,000원 할인이 적용된다
- 오후 8시 이후 상영에 2,000원 할인이 적용된다
- 오후 8시 정각 상영에 2,000원 할인이 적용된다
- 해당하지 않는 시간대에는 할인이 적용되지 않는다

#### 결제 수단

- 신용카드 결제 시 5% 할인이 적용된다
- 현금 결제 시 2% 할인이 적용된다

#### 결제 시스템

- 무비데이 할인이 시간 할인보다 먼저 적용된다
- 포인트가 예매 금액에서 올바르게 차감된다
- 포인트 차감 후 결제 수단 할인이 적용된다
- 신용카드 결제 시 5% 추가 할인이 적용된다
- 현금 결제 시 2% 추가 할인이 적용된다
