# 기능 구현 사항

## 1. 영화 시간 배정

### 영화(Movie)
- [x] 러닝 타임이 같아도 id가 다르면 다른 영화로 판단한다.
- [x] 러닝 타임이 달라도 id가 같으면 같은 영화로 판단한다.

### 러닝타임(RunningTime)
- [x] 러닝 타임이 0 이하일 경우 예외를 던진다.
- [x] 러닝 타임이 양수라면 해당하는 값이 저장된다.
- [x] 러닝 타임의 값이 서로 같다면 동등하다.

### 기간(DateTimeRange)
- [x] 시작 시간이 종료 시간보다 느린 시간이라면 예외가 발생한다.
- [x] 시작 시간이 종료 시간과 같은 시간이라면 예외가 발생한다.
- [x] 2025.12.25일 12시 30분은 2025.12.25 12시 ~ 2025.12.25 13시의 기간에 포함된다.
- [x] 2025.12.25일 11시 30분은 2025.12.25 12시 ~ 2025.12.25 13시의 기간에 포함되지 않는다.
- [x] 2025년 12월 25일 12시는 2025년 12월 25일 12시 ~ 13시의 기간에 포함된다.
- [x] 2025년 12월 25일 13시는 2025년 12월 25일 12시 ~ 13시의 기간에 포함된다.

### 영화 상영 일정(MovieScreening)
- [x] 영화와 기간이 같은 일정은 동일한 일정으로 판단한다.
- [x] 배정된 시간의 길이가 영화의 러닝타임과 일치하지 않으면 예외를 발생시킨다.
- [x] 배정된 시간의 길이가 영화의 러닝타임과 일치하면 예외를 발생시키지 않는다.

### 영화 상영 전체 일정(MovieSchedule)
- [x] 특정 영화의 상영 일정 목록에 포함된 모든 영화들이 같은 영화가 아니라면 예외를 발생시킨다.

### 상영관 일정(ScreenSchedule)
- [x] 운영 기간이 같더라도 id가 다르면 다른 상영관 일정으로 판단한다.
- [x] 운영 기간이 다르더라도 id가 같으면 같은 상영관 일정으로 판단한다.
- [x] 상영관의 운영시간보다 영화 상영 일정이 일찍 배정되면 예외를 발생시킨다.
- [x] 상영관의 운영시간보다 영화 상영 일정이 늦게 배정되면 예외를 발생시킨다.
- [x] 특정 영화를 요청받으면 해당하는 영화 전체 일정(MovieSchedule)을 반환한다.
- [x] 동일한 상영관에서 상영 시간이 겹칠 경우 예외를 발생시킨다.

### 영화관 일정(CinemaSchedule)
- [x] 모든 상영관의 영화 상영 일정을 반환한다.
- [x] 동일한 ScreenSchedule이 들어오면 예외를 반환한다.

## 2. 영화 예매

### 좌석(Seat)
- 좌석 상태 (구매 전, 예약 중, 구매 완료)
- 좌석 등급 생성 (S, A, B)
- [x] 좌석은 열 번호를 가진다. (1 ~ 4)
    - 좌석의 열 번호가 1 미만 4 초과일 경우 예외를 던진다.
- [x] 좌석은 행 문자를 가진다. (A ~ E)
    - 좌석의 행 문자가 F 이상이거나, 2글자 이상일 경우 예외를 던진다.
- [x] 좌석 번호가 다르면 다른 좌석으로 판단한다.
- [x] 좌석 번호가 같으면 같은 좌석으로 판단한다.
      
### 좌석 그룹(SeatGroup)
- [x] 예약중인 좌석에 예약을 시도하면 예약 실패 결과를 반환한다.
- [x] 구매 완료된 좌석에 예약을 시도하면 예약 실패 결과를 반환한다.
- [x] 예약되지 않은 좌석에 예약을 성공하면 예약 성공 결과를 반환한다.
- [x] 중복된 좌석이 존재하면 예외를 던진다.

### 시네마 키오스크(CinemaKiosk)
- [x] 예약한 두 종류 이상의 영화의 시간이 겹치면 예약이 불가능하다.
- [x] 같은 영화 상영 일정에 두 번 이상 예약이 가능하다.

## 3. 결제/할인 (MoviePayment)
- [x] 좌석 등급과 예약한 좌석 수에 따라서 가격이 계산된다.
- [x] 무비데이(매월 10일, 20일, 30일)에 상영되는 영화는 10% 할인된다.
- [x] 오전 11시까지 시작하는 상영은 2,000원이 할인된다.
- [x] 오후 8시부터 시작하는 상영은 2,000원이 할인된다.
- [x] 무비데이이면서 시간 조건을 만족하면 무비데이 할인이 먼저 적용된 후 시간 조건 할인이 적용된다.
- [x] 포인트를 사용하면 예매 금액에서 차감된다.
- [x] 신용카드로 결제하면 포인트 적용 후 금액에서 5% 할인된다.
- [x] 현금으로 결제하면 포인트 적용 후 금액에서 2% 할인된다.
- [x] 할인 조건이 없으면 기본 금액이 그대로 적용된다.
