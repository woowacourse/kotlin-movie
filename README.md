# kotlin-movie

# 기능 목록

## 도메인 - 모델

### 1. 영화

- [x] 영화 제목을 가진다
- [x] 영화 상영 길이를 가진다
- [x] 영화 상영 기간을 가진다

### 1-1. 영화 제목

- [x] 제목을 가진다

### 1-2. 상영 길이

- [x] 분 단위의 상영길이를 가진다

### 1-3. 상영 기간

- [x] 시작일과 종료일을 가진다

### 2. 좌석

- [x] 행과 열을 가진다
    - 행은 알파벳(A~J)으로 이루어져 있다
    - 열은 숫자로 이루어져 있다(1~12)
- [x] 예약 상태를 가진다
- [x] 좌석의 등급을 안다

### 3. 등급

- [x] 등급에 대한 금액을 가진다
    - S석: 18_000
    - A석: 15_000
    - B석: 12_000

### 4. 상영관

- [x] 상영관 이름을 가진다
- [x] 상영관 운영 시간 가진다
- [x] 상영관에 속한 예약 가능한 좌석 목록을 가진다.

### 5. 시간 범위

- [x] 시작, 종료 시간을 가진다
- [x] 시간이 겹치는 지 판단한다

### 6. 상영 정보

- [x] 영화를 가진다
- [x] 상영 날짜 및 시작 시간을 가진다 (`LocalDateTime`)
- [x] 상영관을 가진다
- [x] 상영 시작 및 종료 시간을 가진다
- [x] 해당 상영 회차의 예약 가능 좌석 목록을 관리한다
    - 상영관에 속한 좌석을 기반으로 예약 상태와 함께 초기화한다

### 7. 상영 정보 목록
- [x] 상영 정보를 가진다
- [x] 영화별 상영 목록을 알고있다

### 8. 티켓

- [x] 영화 상영 정보를 가진다
- [x] 예매한 좌석들을 가진다

### 9. 장바구니

- [x] 티켓들을 가진다
- [x] 티켓을 추가한다

### 10. 할인

- [x] 카드 결제 시 할인을 적용한다
- [x] 현금 결제 시 할인을 적용한다
- [x] 무비데이 조건에 해당하는 경우 할인을 적용한다
- [x] 특정 시간대 조건에 해당하는 경우 할인을 적용한다

### 10. 결제 시스템

- [x] 할인 정책을 알고있다.
- [x] 결제 정보를 받아서 총 결제 금액을 계산한다
    - 장바구니
    - 결제 수단(카드, 현금)
    - 포인트

## View

### 예매 시작

- [ ] 예매 시작 안내 메세지를 출력한다
- [ ] 새 예매 생성여부를 입력받는다(Y/N)

### 영화 선택

- [ ] 예매할 영화 제목을 입력받는다

### 예매 날짜 입력

- [ ] 예매 날짜를 입력받는다 (yyyy-MM-dd)

### 예매할 영화 선택

- [ ] 예매할 영화의 상영 번호을 입력받는다

### 예매 좌석 선택

- [ ] 좌석 배치도를 출력한다
- [ ] 예약할 좌석을 입력받는다 (${Row}${Column})

### 장바구니 출력

- [ ] 장바구니에 추가된 예매 정보를 출력한다

### 추가 예매 여부 입력
- [ ] 추가 예매 여부를 입력받는다(Y/N)

### 결제
- [ ] 사용할 포인트를 입력받는다
- [ ] 결제 수단을 입력받는다
  - `1`: 신용카드
  - `2`: 현금
- [ ] 총 결제 금액을 출력한다
- [ ] 예매 확정 여부를 입력받는다
- [ ] 예매 결과를 출력한다

## 테스트 목록

### 영화 테스트

- [x] 영화 제목이 공백일 경우 예외를 던진다
- [x] 영화의 상영 길이가 0이하일 경우 예외를 던진다
- [x] 종료일이 시작일보다 앞 설 경우 예외를 던진다
- [x] 영화는 제목, 상영 길이, 상영 기간을 가진다

### 좌석

- [x] 행과 열을 가진다
- [x] 좌석의 초기 예약 상태는 예약 가능한 상태이다
- [x] 좌석 위치에 따른 등급을 안다
- [x] 행이 A~J 범위가 아닐 경우 예외를 던진다
- [x] 행이 알파벳이 아닌 문자가 포함될 경우 예외를 던진다
- [x] 열이 1~12 범위가 아닐 경우 예외를 던진다

### 좌석 등급

- [x] 등급에 대한 가격을 가진다
- [x] 행과 열에 대한 등급을 반환한다

### 상영관

- [x] 이름, 운영시간, 좌석 목록을 가진다
- [x] 이름이 공백일 경우 예외를 던진다
- [x] 좌석이 없을 경우 예외를 던진다

### 시간 범위

- [x] 종료시간이 시작시간보다 앞서면 예외를 던진다
- [x] 시간 범위가 겹치지 않으면 false를 반환한다
- [x] 시간 범위가 겹치면 true를 반환한다

### 상영 정보

- [x] 영화, 상영관, 상영 날짜 및 시간을 가진다
- [x] 상영 시작 시간과 영화 상영 길이에 따른 시작 및 종료 시간을 가진다
- [x] 상영 시간이 상영관의 운영시간에 포함되지 않을 경우 예외를 던진다
- [x] 좌석을 예약 시 좌석의 상태가 바뀐다

### 티켓

- [x] 상영 정보와 예매 좌석을 가진다
- [x] 예매 좌석에 중복이 있을 경우 예외를 던진다

### 장바구니

- [x] 티켓들을 가진다
- [x] 티켓을 추가할 때, 상영시간이 겹치면 예외를 던진다
- [x] 티켓을 추가하면 티켓이 추가된 장바구니를 반환한다

### 상영 정보 목록
- [x] 상영 정보 리스트를 가진다
- [x] 한 영화가 동시에 상영될 경우 예외를 던진다

### 할인

- [x] 카드 결제 시 할인 금액을 반환한다
- [x] 현금 결제 시 할인 금액을 반환한다
- [x] 무비데이 조건에 해당하는 경우 할인 금액을 반환한다
- [x] 특정 시간대 조건에 해당하는 경우 할인 금액을 반환한다

### 결제 시스템

- [x] 할인 적용 후 금액이 0보다 작을 경우 예외를 던진다
- [x] 총 결제 금액이 올바르게 계산된다
